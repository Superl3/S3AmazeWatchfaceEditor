# S3AmazeWatchfaceEditor ⌚🎨

> **Amazfit GTR 3 전용 고품질 WYSIWYG 워치페이스 테마 에디터 & 컴파일러**
>
> 웹 에뮬레이터 환경에서 컴포넌트를 드래그 앤 드롭하고, 다양한 럭셔리 실시간 안티앨리어싱 패턴과 파스텔 요거트 컬러 프리셋을 적용한 뒤 원클릭으로 Zeus 빌드를 수행하여 워치에 QR 코드로 즉시 전송할 수 있는 통합 웹 기반 테마 편집 플랫폼입니다.

---

## ✨ 핵심 기능 (Key Features)

### 1. 웹 WYSIWYG 레이아웃 편집기
* **드래그 앤 드롭**: 시간(시, 분), 세로 구분선 및 각종 컴플리케이션 위젯(배터리, 걸음 수, 심박수, 칼로리, 이동 거리, 요일/날짜)을 마우스 드래그로 자유롭게 배치할 수 있습니다.
* **상대적 스냅 가이드라인 (Snapping Guide)**: 위젯을 배치할 때 가상의 센터선 및 인접한 다른 컴포넌트들의 외곽 라인에 맞춰 부드럽게 스냅 자석 효과가 작동하며 가이드라인이 표시됩니다.
* **프로퍼티 인스펙터**: 선택한 위젯의 위치(X, Y), 크기(Size), 컬러 타입을 정밀하게 조정하고 실시간으로 추가/삭제할 수 있습니다.

### 2. 고품질 2x 슈퍼샘플링 안티앨리어싱 배경 패턴 엔진
* **19종의 풍성한 배경 테마**: 기하학적 폴리 메시, 도트, 테크 그리드, 물결(Guilloché Waves)부터 시작해 백합 문양이 굽이치는 로열 바로크 벽지(`baroque`), 고대 럭셔리 그리드(`meander`), 제너레이티브 미로(`maze`) 등 총 19종의 고급 배경 무늬를 실시간 지원합니다.
* **바둑판식 무한 반복 (Tiled Repeat)**: 패턴 간격을 좁히거나 회전시켜도 화면 모서리에 빈 여백이 남지 않고 캔버스 전체에 바둑판식으로 무한 연결되어 채워집니다.
* **실시간 간격(Spacing), 회전(Rotation), 크기(Scale) 제어**: 마우스 슬라이더 조작 시 60FPS의 실시간 GPU 가속 가이드를 통해 최적의 심미적 밸런스를 빚어낼 수 있습니다.
* **2배 해상도 프리-로테이션 렌더링**: 워치 디바이스의 연산 및 배터리 절약을 위해, 회전과 간격이 조절된 908x908 고밀도 원본에 안티앨리어싱을 먹인 후 454x454 정적 PNG 파일로 자동 구워 컴파일 패키징에 주입합니다.

### 3. 팬톤(Pantone) & 요거트(Yogurt) 파스텔 테마 프리셋
* 네온 색조를 넘어 우유를 머금은 부드러운 우윳빛 요거트 파스텔 톤 6종 테마 프리셋을 원클릭 지원합니다:
  * **Cyber Coral** (클래식 테크 오렌지)
  * **Strawberry Yogurt** (부드러운 파스텔 딸기우유)
  * **Pistachio Cream** (차분한 세이지 피스타치오 그린)
  * **Blueberry Lavender** (은은한 라벤더 블루베리)
  * **Banana Vanilla** (달콤한 바나나 바닐라 크림)
  * **Soft Peach Melba** (상큼하고 따뜻한 살구 복숭아)
* 워치 디스크린 상에서 액티브 테마 6종이 1:1 동기화되며, 화면 중앙의 구분선을 터치하면 테마 컬러가 워치 내에서 순환 전환됩니다.

### 4. 고해상도 구글 폰트 자동 번들러 (Font Auto-Downloader)
* 사용자가 에디터에서 폰트(`Outfit`, `Orbitron`, `Share Tech Mono`, `JetBrains Mono`)를 변경하면, 컴파일 시 백엔드가 자동으로 해당 폰트의 **ExtraBold / Black 등급 원본 TTF 서체 파일**을 구글 폰트 공식 CDN에서 다운로드하여 번들링합니다.
* 이를 통해 워치 내부에서도 웹 프리뷰와 완벽하게 일치하는 묵직하고 시원한 타이포그래피를 구현합니다.

## ⚙️ 실행 방법 (Getting Started)

### 1. 요구사항 (Prerequisites)
* [Node.js](https://nodejs.org/) (버전 16.x 이상 권장)
* [Zepp CLI (Zeus)](https://docs.zepp.com/docs/guides/tools/cli/) 빌드 환경

### 2. 의존성 패키지 설치
프로젝트 루트 디렉토리에서 터미널을 열고 다음 명령어를 실행하여 필요한 패키지를 설치합니다:

```bash
npm install
```

### 3. 워치페이스 에디터 서버 기동
다음 명령어를 통해 Express 웹 서버를 구동합니다:

```bash
node server.js
```
서버가 성공적으로 실행되면 터미널에 다음 메시지가 출력됩니다:
`Watchface Theme Editor Server running on http://localhost:3000`

### 4. 웹 브라우저 접속 및 편집
1. 웹 브라우저를 열고 **[http://localhost:3000](http://localhost:3000)**에 접속합니다.
2. 메인 화면의 워치 다이얼 내 위젯을 마우스 드래그로 배치하고 스냅 정렬을 시도합니다.
3. 좌측 패널의 `Global Settings`에서 배경 스타일, 간격, 스케일, 오프셋, 오파시티, 회전 각도 및 6가지 요거트 테마 프리셋을 자유롭게 테스트합니다.
4. 배치가 완료되면 우측 하단의 **[Save Configuration]**을 누르고, **[Compile & Generate QR]** 버튼을 클릭합니다.
5. 컴파일이 완료되면 화면에 **설치 QR 코드**가 생성됩니다. 스마트폰의 **Zepp 앱** 또는 개발자 전용 **Zepp 플레이어** 앱의 카메라로 QR 코드를 스캔하여 Amazfit GTR 3 기기로 1초 만에 전송 및 실기기 설치가 완료됩니다!
